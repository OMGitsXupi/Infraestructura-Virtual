# WikiRandom: 
Microservicio que te permitirá compartir u obtener una imagen aleatoria.
Se permitirá subir a una base de datos común, una imagen o link. Se podrá pedir una imagen aleatoria.

## Instrucciones
### Instalación
`fab install`
### Usar la aplicación
`fab main`
### Testear (desarrollo)
`fab test`
